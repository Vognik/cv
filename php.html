Public Transport maps

This application draws route maps for public transportation in Ukraine, Moldova, Bulgaria and others.

Based on data from the database of online service eway.in.ua and constantly updated.

App dynamically downloads OpenStreetMap tiles as it's background. All other elements application draws only relying on PHP functions. For now I'm using GD Graphics library.

Considering that it's not open-source project and I don't even own it, I'm not sure how detailed my narrative should be.

But I'm definitely certain that everyone can use it's results for free to make public transportation more efficient and comfortable.

You can find maps here: static-maps.easyway.info

Open-source application intended to convert numbers into words

Allows easily convert integers between -999 999 999 and 999 999 999 into words in English, Ukrainian or Russian.

There are 3 main classes (1 class for each language), to convert number you should use convertNumber method:

echo ConvertToEnglish::convertNumber(1234);
// outputs "one thousand two hundred and thirty-four"

echo ConvertToUkrainian::convertNumber(1234);
// outputs "одна тисяча двісті тридцять чотири"

echo ConvertToRussian::convertNumber(1234);
// outputs "одна тысяча двести тридцать четыре"

Source code: github.com/Vognik/NumberToText
